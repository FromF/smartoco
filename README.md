# smartoco

## 説明
ESP8266と1.8inch TFTを組み合わせたカウントダウンタイマー。  
ESP8266の基板上にあるPUSHスイッチでカウントダウンを一時停止できます。  
TFTは参考サイトに習って修正が必要です。
